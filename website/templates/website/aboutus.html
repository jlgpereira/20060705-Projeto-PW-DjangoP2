Caraterísticas Elementos/técnicas Exemplos
Layouts Utilização de CSS Grid e CSS flexbox para construção dos layouts das páginas
Responsividade Para a responsividade foi utilizada uma media query que adapta o tamanho da janela do browser consoante seja PC ou Smartphone. @media (max-width: 600px)
Seletores O site tem implementados os selectores universal, tipo, classe, id, atributo, pseudo-classe e pseudo-elemento. Universal

Definição global da font-family, box-sizing, padding, margin e scroll-behavior

Tipo

Redefinição de elementos tais como body, header, nav, main, aside, footer

Classe

Definição de classes tais como .main, .form-container, .header, .menu, .flex, .flex>article, .container_img, .picture_transition, .picture_transition:hover,.welcome-content, .container, .grid_gallery, .grid_img

Id

Definição do id #homepage, utilizado para estilizar o body da Página Princial, que tem um layout específico

Atributo

Utilizado para estilizar as âncoras com target="_blank"

Pseudo-classe

Utilizado para definir a pseudo-classe hover das ancoras

Pseudo-elemento

Utilizado para estilizar o último filho da classe .menu (i.e. a âncora). Combinado com a a pseudo-classe hover e a propriedade visibility permite criar o efeito de seleção de página no menu de navegação de páginas.
